Hallo, 
In den nächsten Monaten werde ich versuchen, eine automatische
Email Archivierung in das DMS von C.U.O.N. zu integrieren. 

Hierfür benötigt der  Server  nun auch das Modul
python-twisted-mail. Dieses Modul stellt einen Zugriff auf einen
Imap-Server bereit, die Konfiguration erfolgt über die server.ini
mit folgenden Zeilen:


#Mail Config zum Archivieren der mails
[IMAP]
SERVER: mailxxx.com
USERNAME: MyName?
PASSWORD: secret
PORT: 143

Da die Client Aktualisierung automatisch über die Server erfolgt,
sollten die Admins C.U.O.N. über das SVN aktualisieren, und dann
mit dem setup tool ein komplettes Update fahren.
