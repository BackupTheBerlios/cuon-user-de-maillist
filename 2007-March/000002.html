Hallo, 
die neue Version von C.U.O.N. steht zum download bereit.

Hier die neuen Features der Version 0.36-0 (Monrovia) von CUON:
Im Kontaktmanagement ist jetzt bei SLEEPS wahlweise Uhrzeit oder Minuten anzugeben

Die Adressverwaltung mausert sich zu einer kleinen Groupwarelösung, so ist es jetzt z. B. möglich, 
den Terminkalender auch über Evolution, Kontact, SUNBIRD sowie anderen Clients anzuschauen, 
zu drucken, etc. Es wird zur Zeit bei jedem Termin jeweils ein Eintrag im Kalender des CUON-Benutzer
und desjenigen, für den der Termin ist, hinzugefügt bzw. geändert.
Dabei werden auch Notizen sowie der Status mit übertragen.

Emails können an Adressen aus CUON heraus gesendet werden, Newsletter an verschiede Gruppen 
gesendet werden. Ebenfalls kann jetzt über CUON auf einen Hylafax-Server ein FAX verschickt werden.
Verschiedene Anredefelder werden bei Briefen in OO berücksichtigt.

Bei den Einstellungen ist jetzt eine Farbwahl für die Eingabefelder möglich.Außerdem kann jetzt bei
Rechnungen, Lieferscheinen und Abholbelegen wahlweise eine Drucker-Option angegeben werden, 
so das diese Belege sofort  zum Drucker geschickt werden.

Bei entsprechender Einstellung werden im Cuon-Hauptfenster alle Termine angezeigt, man kann über 
Buttons sofort zu einem Termin springen.

Viele kleine Änderungen haben das Feeling und Handling verbessert, auch bei großen 
Datenmengen ( 200.000 bis 5000.000 Datensätzen) werden die Daten  schnell angezeigt. 

Zusätzliche Felder wie Briefanreden, Hompage, Notizen runden das Bild ab. 

Als nächstes großes Gebiet wird jetzt die Auftragsverwaltung überarbeitet, anschließend kommt die
Buchhaltung dran.

Ebenfals steht ein vmplayer-Image zum Download bereit, das einen komplett aufgesetzten C.U.O.N.-Server enthält.
Damit kann jeder einfach C.U.O.N. benutzen. Das VM-Image ist keinesfalls eine Notlösung, sondern ist im praktischen Einsatz 
angetestet. Bei kleineren Fa. oder Privatleuten halte ich das für eine gute Sache.

Die Clients sind inzwischen auf mehreren Windows-Rechnern im einsatz und funktionieren dort
ebenfalls. Kleinere Einschränkungen ( z.B. kein Scannen möglich, etc ) sind nicht weiter schlimm. 

Download ist wie üblich über www.cuon.org oder berlios.de möglich.
